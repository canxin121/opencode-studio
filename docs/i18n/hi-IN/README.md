## Quick Start (2 minute)

1. OpenCode CLI install karo (koi ek method):

```bash
# macOS / Linux (official install script)
curl -fsSL https://opencode.ai/install | bash

# macOS / Linux (Homebrew)
brew install anomalyco/tap/opencode
```

```powershell
# Windows (Scoop)
scoop install opencode

# Windows (Chocolatey)
choco install opencode

# Any platform with Bun
bun add -g opencode-ai@latest
```

2. Verify karo:

```bash
opencode --version
```

3. Open in browser:
- Frontend-enabled install: `http://127.0.0.1:3210`
- API-only install: `http://127.0.0.1:3210/health`
